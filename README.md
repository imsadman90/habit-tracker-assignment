Habit Tracker Web App

Live Site: https://habit-tracker-app-1e862.web.app/

A modern web application to help users create, track, and manage their daily habits, build streaks, and boost productivity.

---

Features

Created and Manage Habits : Add habits with title, description, category, reminder time, and optional image.  
Track Streaks: Mark habits as complete and visualize your daily streaks for motivation.  
Browse Public Habits : Explore habits shared by other users for inspiration.  
Responsive Design: Fully mobile-friendly and works on all screen sizes.  
Authentication & Security: Login/Signup using Email/Password or Google account; private routes for your habits.  
Animations & UI Enhancements : Smooth animations using Framer Motion and interactive hero banner carousel.  

---

Tech Stack

Frontend: React, Tailwind CSS, Framer Motion, React Router  
Backend: Node.js, Express.js, MongoDB  
Authentication: Firebase Auth (Email/Password + Google Login)  
Deployment: Firebase (Client)
